Ruta de Integración de WhatsApp

Existen dos formas principales de avanzar con WhatsApp. ¿Cuál encaja mejor con vuestras necesidades?

A

Automatización "Semiautomática" (Sin Coste)

Mejorar el sistema actual con plantillas inteligentes y disparadores automáticos que abren WhatsApp Web con el mensaje ya listo para enviar.

B

API Oficial (Meta / Twilio)

Mensajes enviados 100% en segundo plano sin intervención del agente. Permite chatbots y confirmaciones automáticas de visita (requiere costes de API).
