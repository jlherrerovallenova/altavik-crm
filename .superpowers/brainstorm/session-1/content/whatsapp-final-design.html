Diseño Técnico: Sistema Híbrido de WhatsApp

Aprobación de arquitectura y flujo de datos

1. Estructura de Datos

Tabla: whatsapp_templates
- id: uuid
- name: string (ej: "Seguimiento Post-Visita")
- body: text (con variables: {nombre}, {fecha})
- category: 'marketing' | 'system'
- is_active: boolean

2. El "Motor" de Plantillas

Crearemos un useWhatsAppTemplates que:

  • Carga las plantillas fijas del código (Seguridad 100%).
  • Descarga las plantillas personalizadas de Supabase.
  • Procesa automáticamente variables como el género detectado (interesado/a) y el saludo horario (buenos días/tardes).

3. Flujo de Usuario

Botón "WhatsApp" → Selector de Plantilla → Previsualización → Abrir WhatsApp Web
